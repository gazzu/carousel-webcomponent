# Carousel WebComponents Project

Carousel Web Component with image lazy loading
Created using LitElement and LitHTML

## Instructions

Install polymer cli

```
npm install -g polymer-cli
```

Install the node modules

```
npm install
```

## run & build  

```
npm run start
```

```
npm run build
```

```
npm rum servebuild
```
